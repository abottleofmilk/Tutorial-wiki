
## 标准对话框
Qt为应用程序设计提供了一些常用的标准对话框，如打开文件对话框、选择颜色对话框、信息提示和确认选择对话框、标准输入对话框等，用户无需再自己设计这些常用的对话框，这样可以减少程序设计工作量。
+ QFileDialog文件对话框
+ QFontDialog字体对话框
+ QInputDialog输入对话框
+ QcolorDialog颜色对话框
+ QMessageBox消息框
